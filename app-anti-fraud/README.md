## `app-anti-fraud`


### `Up App`

```bash
# copy .env.example file to .env
$ cp .env.example .env

# install dependency
$ yarn install

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

### `Local App:`
| Resource            | Url                                                                                        |
|---------------------|--------------------------------------------------------------------------------------------|
| app-transaction     | [http://localhost:4000](http://localhost:4000)                                             |
